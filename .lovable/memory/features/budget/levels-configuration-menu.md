# Memory: features/budget/levels-configuration-menu
Updated: now

Configuración de Niveles: El menú "Niveles" unifica la gestión de niveles y todos los parámetros del plano (antes separados en "Parámetros"). Incluye: parámetros generales (espesores, tipo tejado, alero, escala), y configuración por nivel expandible con altura de paredes. Para niveles "bajo cubierta", se muestra un editor dual de pendiente (grados) ↔ altura de cumbre (m), con cálculo bidireccional automático basado en el semi-ancho del edificio. El botón "Guardar y aplicar todo" persiste cambios y reclasifica paredes.
