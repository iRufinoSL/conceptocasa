# Memory: features/budget/levels-configuration-menu
Updated: now

Configuración de Niveles: El menú "Niveles" unifica la gestión de niveles y todos los parámetros del plano (antes separados en "Parámetros"). Incluye: parámetros generales (espesores, tipo tejado, alero, escala), y configuración por nivel expandible con altura de paredes. Para niveles "bajo cubierta", se muestra un editor dual de pendiente (grados) ↔ altura de cumbre (m), con cálculo bidireccional automático basado en el semi-ancho del edificio. Bloques diferenciados por tipo de pared: exterior (largo, alto, espesor) e interior (largo, alto, espesor) con valores independientes configurables en el panel de escala. El botón "Guardar y aplicar todo" persiste cambios y reclasifica paredes.
