# Memory: features/ui-pattern/pdf-coordinate-clipping-fix
Updated: now

Las líneas de cota se renderizan como **divs HTML absolutamente posicionados** (no SVG) para garantizar compatibilidad perfecta con html2canvas en exportaciones PDF. Cada línea de cota consta de 4 divs: línea principal, dos ticks en los extremos, y etiqueta con fondo azul. Las coordenadas usan el mismo sistema que los rooms (container-relative con COL_HEADER_W y ROW_HEADER_H). El contenedor tiene márgenes generosos (marginLeft: COL_HEADER_W+120, marginTop: ROW_HEADER_H+100) para acomodar las cotas superiores/izquierdas que tienen posiciones negativas. DIM_OFF_BOTTOM = DIM_OFF_OUTER + 20, DIM_OFF_RIGHT = DIM_OFF_OUTER + 16 para evitar solapamiento con líneas exteriores. Durante la captura html2canvas se fuerza overflow:visible temporalmente.
