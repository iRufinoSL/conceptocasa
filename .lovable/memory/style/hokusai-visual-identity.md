# Memory: style/hokusai-visual-identity
Updated: now

La identidad visual de la aplicación se basa en la estética ukiyo-e de Katsushika Hokusai, utilizando una paleta de tonos pastel con 3 colores principales:
- **Azul Índigo** (Primary: hsl(210 45% 52%)) — navegación, enlaces, títulos
- **Coral Suave** (Accent: hsl(12 55% 65%)) — acentos secundarios
- **Naranja Hokusai** (Orange: hsl(28 75% 58%)) — estadísticas, CTAs, números de proceso, datos destacados
- **Crema Washi** (Background: hsl(38 35% 96%)) — fondo general

Tipografía: 'Cormorant Garamond' (display/serif) + 'Nunito Sans' (body). Gradientes: gradient-accent (naranja→coral→índigo), gradient-orange (naranja→coral), gradient-warm (crema cálido).
